mvn clean install               // RUN THIS FIRST

mvn -pl server exec:java        // RUN SERVER
mvn -pl client javafx:run       // RUN CLIENT

Run program on 1 computer:
SERVER_IP = "127.0.0.1" in Config.java

Multiple computers:
Use correct IP: (After any IP change --> RUN "MVN CLEAN INSTALL")
Check that IP adress is correct in /shared --> Config.java file.

Find IP adress on your PC:
Windows terminal: "ipconfig" --> find "IPv4 Address" like "192.168.0.15" and insert that
MACOS   terminal: "ifconfig" --> find "IPv4 Address" like "192.168.0.15" and insert that

Testing:
"cd client; mvn clean test"     (frontend tests)
"cd server; mvn clean test"     (backend tests)

Projects fixes:
cntrl + shift + p:
"Maven: Reload Projects"
"Java: Clean Java Language Server Workspace"   -> reload Language

Prerequisites
-------------
- Install JDK 17
- Install Apache Maven ("mvn" on PATH)

Check in PowerShell
-------------------
- java -version
- mvn -version

RUN PROGRAMS
------------
Open two terminals (recommended): one for server, one for client.

Server (from repo root):
- mvn -pl server exec:java

Client (from repo root):
- mvn -pl client javafx:run

Alternative (run from module folders):
- cd server; mvn exec:java
- cd client; mvn javafx:run
